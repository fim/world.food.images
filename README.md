# Cookery Recipe Images

This repository contains recipe images for the [Cookery](../cookery) project.

## Usage

This repository is used as a git submodule in the main Cookery project.

```bash
# Clone the main repository with submodules
git clone --recurse-submodules <main-repo-url>

# Or if already cloned, initialize submodules
git submodule update --init --recursive
```
